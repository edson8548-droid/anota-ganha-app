# Atendimento WhatsApp Venpro

Use esta pasta para guardar duvidas, conversas reais e aprendizados do suporte.

Objetivo:
- registrar o que os representantes perguntam;
- melhorar respostas rapidas;
- encontrar problemas recorrentes;
- preparar uma base real para um futuro agente de atendimento da Venpro.

## Estrutura

- `conversas/`: conversas por dia, com contexto e resposta usada.
- `duvidas/`: resumo de duvidas frequentes por tema.
- `base-agente/`: regras, tom de voz e respostas aprovadas para treinar/configurar o agente no futuro.

## Regra importante

Antes de salvar conversa aqui, remova ou mascare dados sensiveis do cliente:
- nome completo, se nao for necessario;
- telefone;
- email;
- CNPJ;
- endereco;
- tabela de preco;
- dados de pagamento;
- prints com informacoes privadas.

Use identificadores simples como `Cliente 01`, `RCA 02` ou `Empresa A`.
