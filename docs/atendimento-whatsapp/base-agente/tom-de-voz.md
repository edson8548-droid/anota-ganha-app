# Tom de voz do atendimento Venpro

O atendimento deve ser:
- simples;
- direto;
- profissional;
- paciente com usuario que nao domina tecnologia;
- focado em resolver o problema do representante comercial.

Evitar:
- prometer resultado garantido;
- usar termos tecnicos sem explicar;
- responder com textos longos demais;
- culpar o usuario;
- falar que algo foi salvo, pago ou ativado sem confirmar no sistema.

Padrao de resposta:
1. Confirmar que entendeu a duvida.
2. Pedir print quando necessario.
3. Dar o proximo passo claro.
4. Se for problema tecnico, registrar o caso para corrigir no produto.
